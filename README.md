Git & GitHub workflow
=====================

Round 1
-------
1. Create new local repository (one time)

		git init

2. Provide user info to git repo (one time)
		git config --global user.name "your name"
		git config --global user.email "your email"

2.a		Check status

		git status
		
3. Adding files/folders to the staging/indexing
		
		git add -A   --> add all files and folders to staging
		git add filename
		git add *.java
		git add foldername
		
4. Commit the code into Local(git) repository

		git commit -m "commit message"
		
5. Connect local repository with remote repository (one time)
		git remote add origin "https://github.com/fdiouf/CorneropsDataDrivenTestsForBuildersRisk.git"
		
6. Push the code into remote repository

	git push origin master
